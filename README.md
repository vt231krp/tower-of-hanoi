# Tower of Hanoi

Interactive Tower of Hanoi puzzle built with React and TypeScript. Configure game settings, play the challenge, and track your results.

## Configuration

- Node.js: 18+
- Package manager: npm
- Build tool: Vite
- TypeScript config: `tsconfig.json`, `tsconfig.app.json`
- Storybook config: `.storybook/`
- TypeDoc config: `typedoc.json`

## Scripts

- Install dependencies: `npm install`
- Run dev server: `npm run dev`
- Build: `npm run build`
- Preview build: `npm run preview`
- Lint: `npm run lint`
- Generate docs: `npm run docs`
- Run Storybook: `npm run storybook`
- Build Storybook: `npm run build-storybook`

## Documentation

API documentation is generated with [TypeDoc](https://typedoc.org/) and output to `public/docs/`.

## Storybook

Component stories are available for `Button` and `Tower` components. Run `npm run storybook` to view at `http://localhost:6006`.
